Syntax

command reference

Priority
!boss → critical · 100 XP
!high → high · 50 XP
!med → medium · 20 XP
!low → low · 5 XP
Tags
#work · #bug · #design → stack as many as needed
Due dates
today · tomorrow → literal day
mon · tue · fri ... → next weekday
3d · 1w · 2w → in N days / weeks
Keyboard Shortcuts
N → focus input
Enter → add task
Ctrl + D → clone last

example

fix login redirect !high fri #work #bug

→ high · due Friday · work, bug
